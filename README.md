# Responsive Landing Page - AutoScale

A modern, high-performance, responsive landing page for a premium toy car model collection. This project has been upgraded with a sleek UI/UX, glassmorphism, smooth animations, and is optimized for deployment.

## Features
- **Modern UI**: Uses advanced CSS techniques like glassmorphism and gradient meshes.
- **Micro-Animations**: Scroll reveals and hover interactions to keep the user engaged.
- **Responsive**: Fully responsive on mobile, tablet, and desktop viewports.
- **FontAwesome Icons**: Lightweight, modern iconography.
- **Vercel Ready**: Structured with an `index.html` at the root for zero-config Vercel deployment.

## Deployment
This project can be easily deployed to [Vercel](https://vercel.com/).
Since the main entry point is named `index.html`, simply import this repository into your Vercel dashboard and deploy it without any custom build commands.

## Local Development
To run this project locally, simply open `index.html` in your browser, or start a local server:
```bash
python -m http.server 8080
# or
npx serve .
```
